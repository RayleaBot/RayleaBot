# v0.1 Changelog

RayleaBot v0.1 已交付的能力清单与边界。本文件作为历史基线归档，保留版本对照与回顾用途。

## 主线交付

- 单实例自托管基线：`raylea-server` 单二进制、Web 管理面、Electron Launcher、CLI 共享同一进程入口。
- 工程治理：根与子目录 `AGENTS.md`、`.agents/skills/`、`docs/engineering/baseline.md`、`docs/engineering/implementation-order.md`。
- Contracts：12 份 fixture-ready formal contracts（`backup-manifest`、`config.user`、`deps-manifest`、`error-codes`、`web-api`、`websocket-events`、`plugin-info`、`plugin-management-ui`、`plugin-management-ui-bridge`、`plugin-protocol`、`release-manifest`、`cli-commands`）。
- Fixtures / Golden Cases：config、web-api、websocket、plugin-info、plugin-protocol、release-manifest、CLI 全部进入 CI 校验。

## Server 内核

- 入口、配置 schema 校验、统一 `slog` 日志、`/healthz`、`/readyz`。
- SQLite：WAL、busy_timeout、迁移序列嵌入二进制、读写句柄分离。
- Auth：bootstrap、login、签名、TTL、滑动续期、并发会话上限、持久化。
- Tasks：异步执行器、历史持久化、跨重启 hydration。
- Plugin discovery：`plugins/builtin/` 与 `plugins/installed/`，`examples/plugins/` 仅示例。

## Adapter / 协议主链

- OneBot11 reverse WebSocket、心跳感知、状态机（`idle / connecting / connected / auth_failed / reconnecting / stopped`）、ready frame gating、退避重连。
- 事件归一化覆盖 `message.private`、`message.group`、`notice.member_increase`、`notice.member_decrease` 等基础事件。
- `message.send` / `message.reply` 出站、`get_login_info` 等内部 API、identity cache。
- forward WebSocket、HTTP API 与 webhook 由后续版本补齐。

## 插件平台

- per-plugin Runtime Manager、`init -> init_ack`、`ping/pong`、`shutdown`。
- Dispatcher fan-out、命令定向、scheduler trigger、zero-gap reload。
- Local action：`logger.write`、`storage.kv`、`storage.file`、`http.request`、`config.read`、`config.write`、`scheduler.create`、`event.expose_webhook`、`render.image`，gated `event.raw_payload`。
- 插件状态：`stopped / starting / running / stopping / crashed / backoff / dead_letter`。
- 临时授权：`expires_at` 进入 grants 管理面与运行时判定。

## 配置与存储

- Canonical config：`schema_version=2`、`config/default.yaml` 默认基线、首份 `user.yaml` 自动 bootstrap。
- 配置热更新：`PUT /api/config` 返回 `apply_effects`，区分立即生效、局部重载与 `restart_required`。
- CLI：`reset-admin`、`backup`、`restore`、`doctor`、`migrate`、`cleanup`。
- 聊天侧治理：黑名单、命令权限、冷却限流。

## 管理面

- HTTP：setup、session、config、system status / shutdown / backup / diagnostics、logs、plugins lifecycle、plugin grants、recovery / runtime tasks、render preview。
- WebSocket：`/ws/events`、`/ws/logs`、`/ws/plugins/{id}/console`。
- Web UI：登录、初始化、状态、插件管理、日志、协议、配置、模板预览，全部基于 Ant Design Vue + Vue Vben Admin 对齐方案。
- Launcher：环境检查、启停、健康轮询、托盘语义、版本检查、设置持久化、安装根目录派生模型。

## 渲染服务

- Chromium 渲染、模板版本仓、artifact registry、`render.preview` 任务模型。
- `.deps/manifest.json` 固定 Chromium / Python / Node.js 资源的有序来源、SHA256、archive_format 与 entrypoints。
- 离线 / 受限网络场景支持 `cache/downloads/runtime/` 与 `.deps/store/<resource-id>/<version>/`。

## 发布与质量

- 正式产物：`windows-x64-full`、`linux-x64-full`、`macos-arm64-full`、`linux-x64-server`。
- Release metadata：`release_manifest.json`、`build_info.json`、`SHA256SUMS.txt`。
- CI：`contracts.yml`、`lint.yml`、`race.yml`、`release.yml`、`self-host-smoke.yml`。
- Recovery：`recovery_summary`、`recovery.recheck`、`recovery.confirm`、`runtime.bootstrap`。
